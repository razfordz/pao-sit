## Project Overview

This project is a GovTech AI platform called “เป๋าสิทธิ์” (Pao-Sit).

“เป๋าสิทธิ์” is an AI-powered welfare assistant designed to help citizens discover, understand, and access government benefits, welfare programs, and public services more easily using AI, Open Government Data, and ThaiLLM for personalized recommendations and eligibility analysis.

System goals:
- Help citizens understand what benefits they are eligible for
- Reduce the complexity of bureaucratic/government language
- Simplify access to welfare and public services
- Make government services more accessible for all age groups

The platform should feel:
- friendly
- easy to understand
- modern
- trustworthy
- accessible for everyone

The platform should NOT feel:
- like traditional government software
- overly complicated
- too technical
- cyberpunk or futuristic

## UX Principles

Prioritize:
- simplicity
- accessibility
- emotional clarity
- readable typography
- minimal user friction
- conversational experience

The UI should:
- be understandable at first glance
- be easy to use for all age groups
- use large buttons
- use simple and friendly Thai language
- minimize unnecessary steps

Avoid:
- cluttered dashboards
- complicated navigation
- small text
- difficult bureaucratic wording

---

## AI Interaction Style

The AI Assistant should feel:
- polite
- friendly
- warm
- easy to understand
- conversational
- like a personal assistant

Use simple conversational Thai instead of formal bureaucratic language.

Example:

Instead of:
“ดำเนินการยื่นคำร้อง”

Use:
“สมัครสิทธิ์”
